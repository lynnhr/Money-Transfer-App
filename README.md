# Money Transfer App â€” BonPay

Cross-platform React Native mobile app for peer-to-peer and domestic transfers. Backend: Node.js + Express + MySQL.

## Key features
- User auth: sign up, login
- Send / receive money, optional message
- Transaction history with filters
- Receive via user ID / QR code
- Support: contact form, FAQ, privacy/terms links

## Tech stack
- Mobile: React Native 
- Backend: Node.js, Express
- Database: MySQL
